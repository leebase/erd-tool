# Project Plan

## Current direction

- Build around a durable canonical physical model.
- Start with Snowflake metadata extraction, editable diagrams, reusable project
  files, DDL generation, and round-trip verification.
- Evaluate open-source foundations such as drawDB and Snowflake metadata tools
  before replacing mature components.
- Use auto-orch for ongoing discovery and Agent-Orch for governed delivery
  slices with smoke and review-verdict gates.
