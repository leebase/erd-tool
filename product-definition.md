# Product Definition

- Project: erd-tool
- Primary language: TypeScript/Python
- Problem statement: Database professionals need a modern, affordable modeling
  studio that can reverse engineer, visualize, edit, document, and forward
  engineer relational schemas without enterprise bloat.
- First production focus: Snowflake reverse engineering and round-trip
  engineering.
- Users: independent consultants, software engineers, database architects,
  educators, and small engineering organizations.
- Success criteria: experienced database professionals choose it over expensive
  modeling tools because it is fast, elegant, accurate, and maintainable.
