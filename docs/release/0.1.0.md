# ERD Tool 0.1.0

Release date: 2026-07-15

ERD Tool 0.1.0 is the first source release of the installable data modeling
application. It brings the proven SQLite-to-Snowflake structural workflow into
an Electron desktop application and adds direct Snowflake reverse engineering.

## Validated release target

- Apple Silicon macOS
- Electron 40
- Node.js 22 development environment
- Unsigned and not notarized

## Acceptance evidence

- The desktop unit and integration suite passes.
- ESLint, web renderer, desktop renderer, and Electron main/preload builds pass.
- The Apple Silicon DMG and ZIP package successfully.
- Production dependency audit reports no known vulnerabilities.
- A live Chinook structure round trip proved 11 tables, 64 columns, 11 primary
  keys, and 11 foreign keys in Snowflake, with zero enforced key constraints.

## Install from source

```bash
git clone https://github.com/leebase/erd-tool.git
cd erd-tool/desktop
npm ci
npm run start:electron
```

## Package locally

```bash
cd desktop
npm run dist:desktop:mac:arm64
```

The generated application is unsigned. Review the source and use Finder's
explicit **Open** action if Gatekeeper blocks the first launch.
